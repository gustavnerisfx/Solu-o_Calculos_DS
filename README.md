### Projeto de aula Menu Calculadora

<p>Este projeto foi feito em sala de aula, escrito em C# .NET Framework, Windows Forms.</p>
<p>O projeto consiste em ser um software com várias ferramentas, como calculadoras, bloco de notas e um navegador.</p>

![imagem do projeto](https://raw.githubusercontent.com/gustavnerisfx/Solu-o_Calculos_DS/refs/heads/master/Captura%20de%20tela%202026-04-06%20115501.png)

<p>O software contém uma calculadora básica, de layout simples e que permite realizar as principais operações, comparar se dois números são ímpar ou par e qual é o maior.</p>
