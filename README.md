### Projeto de aula Menu Calculadora

<p>Este projeto foi feito em sala de aula, escrito em C# .NET Framework, Windows Forms.</p>
<p>O projeto consiste em ser um software com várias ferramentas, como calculadoras, bloco de notas e navegador.</p>
