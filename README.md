## Projeto de aula Menu Calculadora

<p>Este projeto foi feito em sala de aula usando C# Windows Forms.</p>
<p>O projeto consiste em ser uma calculadora, com opções de soma, subtração, divisão e multiplicação; junto com funcionalidades de comparar números e se o resultado é ímpar ou par.</p>

![imagem do projeto](https://github.com/gustavnerisfx/Solu-o_Calculos_DS/blob/master/imgc%23.png)

<p>O Ghost Thundra Browser tem as funcionalidades comum de qualquer outro navegador. Se pode pesquisar pela barra de endereço, voltar para a página anterior ou avançar, e recarregar a página.</p>
